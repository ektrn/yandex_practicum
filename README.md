## Описание проектов 

Данные проекты были выполнены в ходе обучения в Яндекс.Практикуме, профессии "Аналитик данных".

| Название проекта | Описание | Используемые библиотеки | 
| :---------------------- | :---------------------- | :---------------------- |
| [A/B-тестирование](abtest_project) | Изучить воронку продаж. Узнать, как пользователи доходят до покупки. Сколько пользователей доходит до покупки, а сколько — «застревает» на предыдущих шагах? На каких именно? Исследовать результаты A/A/B-эксперимента.| *pandas, matplotlib, scipy, numpy, math, plotly* |
| [Анализ оттока клиентов банка](banks_project) | Максимально эффективно выделить сегменты отточных клиентов, чтобы они были однородными, высокоотточными и покрывали максимум проблемных зон банка | *pandas, matplotlib, scipy, seaborn, plotly* |
