# Описание проекта по анализу оттока клиентов в банке


## Данные

Датасет содержит данные о клиентах банка «Метанпром». Банк располагается в Ярославле и областных городах: Ростов Великий и Рыбинск.

Колонки:

- идентификатор пользователя,
- баллы кредитного скоринга,
- город,
- пол,
- возраст,
- количество объектов в собственности,
- баланс на счёте,
- количество продуктов, которыми пользуется клиент,
- есть ли кредитная карта,
- активный клиент,
- заработная плата клиента,
- ушёл или нет.

## Задача

максимально эффективно выделить сегменты отточных клиентов, чтобы они были однородными, высокоотточными и покрывали максимум проблемных зон банка

## Используемые библиотеки
*pandas*, *matplotlib*, *scipy*, *seaborn*, *plotly*
